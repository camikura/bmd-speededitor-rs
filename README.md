# BMD SpeedEditor
Speed Editor HID implementation in Rust

# Callback

You can see how to use callbacks in the sample.

# Example
You can run it with cargo run:
```
cargo run --package basic
```

You can build it with cargo build:
```
cargo build --package basic
```
